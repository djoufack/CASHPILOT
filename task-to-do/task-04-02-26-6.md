# Task 6 - HAUTE : Corriger localStorage.clear() au logout

## Fichier a modifier
- `src/hooks/useAuth.js` (ligne 29)

## Probleme
`localStorage.clear()` efface TOUTES les preferences utilisateur (langue, sidebar state, etc.) lors du logout.

## Solution
Remplacer `localStorage.clear()` par des suppressions selectves des cles liees a l'authentification uniquement.

## Code attendu
Remplacer la ligne 29:
```js
// AVANT
localStorage.clear();

// APRES
// Clear only auth-related items, preserve UI preferences (language, sidebar, etc.)
const authKeys = ['sb-rfzvrezrcigzmldgvntz-auth-token', 'supabase.auth.token'];
authKeys.forEach(key => localStorage.removeItem(key));
```

Note: Les cles Supabase auth suivent le pattern `sb-{project-ref}-auth-token`. Les preferences comme `language`, `sidebarCollapsed`, `i18nextLng` doivent etre preservees.
