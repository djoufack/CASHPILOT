# Task 17 - BASSE : Cout AI dans useCreditsGuard

## Fichiers a modifier
- `src/hooks/useCreditsGuard.js` (lignes 33-36, 68-69, 99-103)

## Probleme
Le systeme de credits ne connait pas le cout de l'extraction IA de factures.

## Solution
Ajouter `AI_INVOICE_EXTRACTION: 3` dans CREDIT_COSTS, son label dans CREDIT_COST_LABELS, et une nouvelle categorie AI_FEATURES dans CREDIT_CATEGORIES.

## Modifications

### 1. Dans CREDIT_COSTS (apres ligne 35, avant la fermeture `};`)

Ajouter :
```javascript
  // IA
  AI_INVOICE_EXTRACTION: 3,
```

### 2. Dans CREDIT_COST_LABELS (apres ligne 68, avant la fermeture `};`)

Ajouter :
```javascript
  // IA
  AI_INVOICE_EXTRACTION: 'credits.costs.aiInvoiceExtraction',
```

### 3. Dans CREDIT_CATEGORIES (apres la categorie OTHER ligne 102, avant la fermeture `};`)

Ajouter :
```javascript
  AI_FEATURES: [
    'AI_INVOICE_EXTRACTION',
  ],
```

- [ ] CREDIT_CATEGORIES contient AI_FEATURES
