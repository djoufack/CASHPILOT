# Task 001 - [MOYENNE] : Code Splitting React.lazy pour toutes les pages

## Fichier(s) a modifier
- `src/App.jsx` (lignes 1-165)

## Probleme
Actuellement, seulement 4 pages utilisent React.lazy() :
- SecuritySettings
- RecurringInvoicesPage
- BankConnectionsPage
- CashFlowPage

Les autres 25+ pages sont importees statiquement, ce qui augmente le bundle initial et degrade les performances de chargement.

## Solution
Convertir tous les imports de pages en imports dynamiques avec React.lazy() et creer un composant LoadingSpinner reutilisable pour les Suspense fallbacks.
