# Task 7 - HAUTE : Deplacer clef VAPID en variable d'environnement

## Fichier a modifier
- `src/hooks/usePushNotifications.js` (ligne 46)
- `.env.example` (ajouter la variable)

## Probleme
La clef publique VAPID est hardcodee dans le code source.

## Solution
1. Remplacer la valeur hardcodee par `import.meta.env.VITE_VAPID_PUBLIC_KEY`
2. Ajouter `VITE_VAPID_PUBLIC_KEY=your-vapid-public-key-here` dans `.env.example`

```js
// APRES
applicationServerKey: import.meta.env.VITE_VAPID_PUBLIC_KEY
```
