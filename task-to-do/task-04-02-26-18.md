# Task 18 - BASSE : Cles i18n pour l'extraction IA

## Fichiers a modifier
- `src/i18n/locales/en.json`
- `src/i18n/locales/fr.json`
- `src/i18n/locales/nl.json`

## Probleme
Aucune cle de traduction n'existe pour la fonctionnalite d'extraction IA.

## Solution
