# Task 002 - [FAIBLE] : Edge Caching CDN Headers

## Fichier(s) a modifier
- `vercel.json` (root)

## Probleme
Les assets statiques (JS, CSS, images) ne sont pas mis en cache de maniere optimale par le CDN. Cela augmente la latence pour les utilisateurs internationaux.

## Solution
Ajouter des headers Cache-Control agressifs pour les assets immutables (bundles JS avec hash, images, fonts).
