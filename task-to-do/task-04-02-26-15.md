# Task 15 - MOYENNE : Accepter images dans useInvoiceUpload

## Fichiers a modifier
- `src/hooks/useInvoiceUpload.js` (lignes 14-19)

## Probleme
Le hook useInvoiceUpload n'accepte que les fichiers PDF. Les factures scannees (photos JPG/PNG) sont rejetees.

## Solution
Modifier la fonction `validateFile` pour accepter egalement `image/jpeg` et `image/png`, en plus de `application/pdf`.

## Code attendu (remplacement lignes 14-19)

```javascript
  const validateFile = (file) => {
    const allowedTypes = ['application/pdf', 'image/jpeg', 'image/png'];
    const isTypeValid = allowedTypes.includes(file.type);
    const isSizeValid = file.size <= 10 * 1024 * 1024; // 10MB

    if (!isTypeValid) throw new Error("Only PDF, JPG and PNG files are allowed");
    if (!isSizeValid) throw new Error("File size must be less than 10MB");
  };
```
