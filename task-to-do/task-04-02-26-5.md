# Task 5 - HAUTE : Nettoyer les console.log en production

## Fichiers a modifier
- `src/hooks/useAuth.js` (environ 15 console.log/warn/error)
- `src/lib/supabase.js` (environ 5 console.log)

## Probleme
Des dizaines de console.log avec emojis (debug) sont presentes dans le code de production, exposant des informations internes.

## Solution
Dans `useAuth.js` : supprimer tous les `console.log(...)` et `console.warn(...)` de debug.
Garder les `console.error(...)` pour les erreurs reelles mais sans emojis.
Dans `supabase.js` : supprimer les `console.log("[Supabase]...")`.

IMPORTANT: Ne pas supprimer les `console.warn` qui indiquent des problemes RLS - ceux-ci restent utiles.
