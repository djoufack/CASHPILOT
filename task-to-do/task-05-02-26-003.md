# Task 003 - [MOYENNE] : PWA Complete - Offline Page

## Fichier(s) a modifier
- `public/offline.html` (nouveau fichier)
- `public/sw.js` (lignes 1-50)

## Probleme
L'application PWA n'a pas de page offline dediee. Quand l'utilisateur est hors ligne et tente d'acceder a une page non mise en cache, il voit une erreur generique du navigateur.

## Solution
Creer une page offline.html elegante et l'integrer dans le Service Worker pour servir cette page quand le reseau est indisponible.
