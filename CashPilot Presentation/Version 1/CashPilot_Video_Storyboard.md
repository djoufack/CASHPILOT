# CashPilot - Script & Storyboard Video

**Format** : Presentation video / Product Tour
**Duree estimee** : 18-22 minutes (ou decoupable en 10 episodes de 2 min)
**Ton** : Professionnel, accessible, dynamique
**Langue** : Francais
**Public cible** : Entrepreneurs, freelances, PME, comptables

---

## Structure Generale

| Episode | Sujet | Duree |
|---------|-------|-------|
| 1 | Introduction & Premiers Pas | 1:30 |
| 2 | Tableau de Bord | 2:00 |
| 3 | Gestion des Clients | 1:30 |
| 4 | Gestion des Projets & Feuilles de Temps | 2:30 |
| 5 | Facturation | 3:00 |
| 6 | Devis, Bons de Commande & Avoirs | 2:00 |
| 7 | Depenses & Stocks | 2:00 |
| 8 | Fournisseurs & Creances/Dettes | 2:00 |
| 9 | Integration Comptable | 2:30 |
| 10 | Scenarios, Analyses & Parametres | 2:00 |

## EPISODE 1 : Introduction & Premiers Pas

**Duree** : 1 min 30 s

### Scene 1.1 — Ouverture (0:00 - 0:20)

**Visuel** :
- Fond sombre avec particules animees ou gradient bleu/violet
- Le logo CashPilot apparait au centre avec une animation de zoom doux
- Tagline en dessous : "Votre solution complete de gestion d'entreprise"
- Transition : fondu vers la landing page de l'application

**Narration** :
> "Bienvenue dans CashPilot. Facturation, projets, comptabilite, stocks : tout ce dont vous avez besoin pour gerer votre entreprise, dans une seule application. Dans cette presentation, nous allons parcourir ensemble chaque fonctionnalite."

**Musique** : Intro dynamique corporate, moderne (style tech startup)

**Annotations ecran** : Aucune

- Screencast : Page de connexion de CashPilot

- Remplissage accelere (motion design) des champs

**Narration** :
> "Pour commencer, connectez-vous avec votre adresse email et votre mot de passe. Si vous n'avez pas encore de compte, cliquez sur S'inscrire, remplissez le formulaire, validez votre email, et configurez votre profil d'entreprise. C'est fait en quelques minutes."

## EPISODE 5 : Facturation

### Scene 5.2 — Creer une facture (0:30 - 1:20)

**Visuel** :
- Clic "+ Nouvelle Facture"
- Formulaire Mode Standard :
  - Informations generales : numero (auto), client, dates
  - Section Lignes de facture : ajout d'une ligne (Description, Quantite, Prix unitaire, TVA, Montant auto)
  - Ajout d'une 2e ligne
  - Notes et conditions
- Toggle vers Mode Rapide : formulaire simplifie
- Retour Mode Standard
- Clic "Apercu" : modal avec la facture formatee

Pour chaque ligne, saisissez la description, la quantite, le prix unitaire et le taux de TVA. Le montant se calcule automatiquement. Ajoutez des notes et conditions. Le mode Rapide simplifie le formulaire pour une creation acceleree. Avant de valider, utilisez l'apercu pour verifier le rendu."

**Annotations ecran** :
- Encadre "Mode Standard" / "Mode Rapide" sur le toggle

## Notes de Production

### Style Visuel Recommande

| Element | Specification |
|---------|---------------|
| **Resolution** | 1920x1080 (Full HD) minimum |
| **Ratio** | 16:9 |
| **Police titres** | Inter Bold ou Poppins Bold |
| **Police corps** | Inter Regular |
| **Couleur principale** | Bleu #3B82F6 (style CashPilot) |
| **Couleur secondaire** | Violet #8B5CF6 |
| **Couleur accent** | Vert #10B981 |
| **Couleur alerte** | Rouge #EF4444 |
| **Fond ecran** | Blanc #FFFFFF ou Gris clair #F9FAFB |
| **Fond sombre** (intro/outro) | #0F172A |

### Effets et Animations

- **Transitions** : Alternance entre fondu, slide, wipe (pas toujours le meme)
- **Highlights ecran** : Encadres jaunes semi-transparents (60% opacite)
- **Fleches** : Rouges ou blanches, animees (apparition progressive)
- **Tooltips** : Bulles arrondies, fond sombre, texte blanc
- **Badges** : Pilules arrondies avec couleur de fond
- **Compteurs** : Animation de chiffres qui montent (easing ease-out)
- **Zoom** : Smooth zoom avec duree 0.5s

### Musique et Son

| Moment | Style |
|--------|-------|
| **Intro/Outro** | Corporate moderne, piano + beat electronique leger |
| **Pendant narration** | Musique de fond tres discrete (volume 10-15%) |
| **Transitions** | Petit swoosh ou clic discret |
| **Highlights** | Son "pop" leger quand un element est mis en evidence |

### Voix Off

- **Ton** : Professionnel mais chaleureux, pas robotique
- **Debit** : Modere (150-160 mots/minute), pauses aux transitions
- **Genre** : Voix masculine ou feminine, au choix (les deux fonctionnent)
- **Langue** : Francais metropolitain, articulation claire

### Outils Recommandes

| Usage | Outils |
|-------|--------|
| **Screencast** | OBS Studio, Loom, ScreenFlow |
| **Montage** | DaVinci Resolve (gratuit), Premiere Pro, Final Cut |
| **Motion design** | After Effects, Motion (Mac), Canva Video |
| **Voix off** | Enregistrement studio ou ElevenLabs / PlayHT (IA) |
| **Musique** | Artlist, Epidemic Sound, YouTube Audio Library (gratuit) |
| **Miniature** | Canva, Figma |

### Declinaisons Possibles

1. **Video complete** (18-22 min) : Enchainer les 10 episodes
2. **10 episodes courts** (2 min chacun) : Publier separement (YouTube, site)
3. **Version courte** (3-5 min) : Episodes 1, 2, 5, 10 uniquement (teaser)
4. **GIF animes** : Extraire les demos cles pour documentation / marketing
5. **Version sous-titree** : Ajouter sous-titres FR et EN
