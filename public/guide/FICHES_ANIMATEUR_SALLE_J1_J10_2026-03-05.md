# Fiches animateur en salle - CashPilot (J1 a J10)

Date: 2026-03-05  
Usage: support imprime pour animation quotidienne en presentiel/distanciel  
Reference detaillee: `docs/guide/COURS_FORMATEUR_JOURNALIER_COMPLET_2026-03-05.md`

## Mode d'emploi (formateur)

- Imprimer 1 section par jour.
- Cocher les checkpoints en direct.
- Collecter les artefacts de fin de journee.
- Noter les blocages pour le debrief du lendemain.

---

## Fiche J1 - Navigation, contexte, securite

### Objectif du jour

- Savoir se reperer dans tous les menus.
- Verifier societe active et securite compte.

### Timing

- 30 min recap + quiz
- 60 min demo
- 90 min atelier guide
- 90 min atelier autonome
- 30 min correction

### Script demo

1. Dashboard: lecture KPI en 5 min.
2. Pilotage: lecture signal -> decision.
3. Portefeuille societes: comparaison rapide.
4. Parametres/Securite: baseline compte.

### Exercice eleve

- Produire un memo: "3 alertes + 3 actions".

### Checkpoints formateur

- [ ] Eleve choisit la bonne societe active
- [ ] Eleve distingue Dashboard vs Pilotage
- [ ] Eleve sait ou regler securite

### Artefacts a remettre

- Capture dashboard annotee
- Memo 3 alertes/3 actions

- 1 controle finance

- [ ] Reporting pertinent
- [ ] Controle finance correct
- [ ] Integration fonctionnelle

### Artefacts

- Grille notee
